How to run:
1. Open the folder in VS Code.
2. Use a simple local server (for example, the Live Server extension in VS Code).
3. Start with index.html.

Why a local server matters:
The quiz uses fetch/AJAX to load questions from data/questions.json.
Most browsers block local fetch requests if you open the page directly as a file.
